# 🎮 像素大乱斗：谁能活到最后？

一款魔性搞笑的 2D 像素风射击游戏，7 位好友相爱相杀，谁能笑到最后？

## 🎯 游戏特色

- **7位独特角色**：每个角色都有独特的血量、移速、武器和专属技能
- **魔性技能**：画大饼、删库跑路、奶茶轰炸、萌力咆哮...各种沙雕技能等你体验
- **高清像素风**：HD-2D 质感，现代光影特效，发光子弹、粒子爆炸
- **相爱相杀局**：选择你的角色，在**3分钟**内干翻其他6个"朋友"
- **平衡战斗**：精心调整的数值，既不会太简单也不会太困难

## 🕹️ 操作说明

| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 射击 | 鼠标瞄准 + 点击 |
| 技能 | 空格键 |

**游戏目标**：在3分钟内击败全部6个敌人！

## 🚀 运行方式

### 方式1：直接打开（推荐）
```bash
# 用浏览器直接打开 index.html
open index.html
```

### 方式2：本地服务器
```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 📁 项目结构

```
pixel-brawl-game/
├── index.html              # 游戏入口
├── README.md               # 本文件
└── src/
    ├── main.js             # Phaser配置
    ├── data/
    │   └── characters.js   # 角色数据
    ├── entities/
    │   ├── Bullet.js       # 子弹类
```

## 📝 开发说明

### 添加新角色
在 `src/data/characters.js` 中添加角色配置：

```javascript
newchar: {
    id: 'newchar',
    name: '新角色',
    hp: 100,
    speed: 200,
    color: 0xFF0000,
    emoji: '🎮',
    weapon: { ... },
    skill: { ... }
}
```

### 调整游戏参数
在 `GameScene.js` 中修改：
- `this.gameTime`：游戏时长（默认**180秒=3分钟**）
- 敌人强度：在 `Enemy.js` 中调整血量倍数和AI侵略性
